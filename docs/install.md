# インストール手順

あらかじめPCに「Aviutl」と「拡張編集」を導入しておいてください

Aviutlと拡張編集が構築できましたら  
同梱のAviutl100フォルダの中にある

- script

このスプライトスクリプトデータを  
お使いのPCのAviutlフォルダの直下にコピーしてください。  
（Aviutl.exeファイルと同じ階層にコピーします）

例えば以下のようになっていれば正常です。
> Aviutl100 ［フォルダ］
>> Aviutl.exe ［exeファイル］  
>> script ［フォルダ］
